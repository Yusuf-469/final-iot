# Dr. AI 3D Model Setup

## GLB Model File

To use your custom 3D medical doctor model:

1. Copy your GLB file to `frontend/models/medical-doctor-3d-model.glb`
2. The file should be named exactly: `medical-doctor-3d-model.glb`
3. The 3D model will load automatically when you visit the Dr. AI page

## Features

- **Interactive 3D Model**: Click the model to open the chat interface
- **Fallback Model**: If GLB fails to load, shows a procedural sci-fi model
- **Smooth Transitions**: Seamless switching between 3D view and chat
- **Real-time Animation**: Gentle floating and rotation animations
- **Professional Lighting**: Multiple light sources for realistic rendering
